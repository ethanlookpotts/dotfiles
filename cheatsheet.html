<SPC>fr — Recent files
<SPC>fb — File browser

Code Nav

gd — Go to definition
gr — Show references
K — Hover docs
<SPC>ca — Code actions
<SPC>rn — Rename

Diagnostics

[d — Previous diagnostic
]d — Next diagnostic
<SPC>db — Buffer diagnostics
<SPC>dl — Line diagnostics

Git

<SPC>gd — Diff view
<SPC>gp — Preview hunk
<SPC>gb — Blame
<SPC>gs — Stage hunk
<SPC>gr — Reset hunk

Format

<SPC>mp — Format
<SPC>mc — Format changed
<SPC>ml — Lint

gcc — Toggle comment

NEOVIM

Movement

Command Description

0 / $Move to start/end of line
gg / GGo to start/end of file

Editing

Command Description

Command Description
